# Example Skill

このファイルは、ClaudeCodeスキルのテンプレートです。

## 説明

このスキルは例として提供されています。外部から入手したskills.mdファイルは、同様の構造でこのディレクトリに配置してください。

## 使い方

1. 新しいスキル用のディレクトリを作成: `skills/<skill-name>/`
2. スキルファイルをコピー: `skills/<skill-name>/skill.md`
3. `marketplace.json`に登録情報を追加

## スキルファイルの構造

スキルファイル（skill.mdまたはskills.md）には、通常以下のような情報が含まれます：

- スキルの説明
- 使用方法
- パラメータ
- 実行例
- 依存関係

## 注意事項

- ファイル名は `skill.md` または `skills.md` のどちらでも構いません
- 各スキルは独立したディレクトリに配置することを推奨します
- バージョン管理のため、`marketplace.json`を更新してください
